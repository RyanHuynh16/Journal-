HTML Colors

HTML color is a code that can be added into a <h1>-<h6> or <p>. HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values. In HTML, a color can be specified by using a color name (tomato, orange, dodger blue, medium sea green).
